# Genre Presets

## sci-fi
- Тон: технологическая достоверность + человеческая драма.
- Темп: средний, с постепенным наращиванием ставок.
- Детали: инфраструктура, устройства, следствия решений.

## urban-fantasy
- Тон: повседневный город + скрытая магическая изнанка.
- Темп: средне-высокий, ритм сцен через открытия и конфликты.
- Детали: кварталы, привычные места, тайные правила слоёв города.

## mixed
- Сочетай sci-fi и urban-fantasy без жанрового конфликта.
- При противоречии приоритет у уже сохранённого `world.json`.
