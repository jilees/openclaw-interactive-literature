# Story Contract (generation output)

Генерируй в этом формате (внутренний контракт):

```json
{
  "scene_text": "4-6 абзацев литературного текста на русском",
  "continuity_notes": [
    "короткая проверка консистентности #1",
    "короткая проверка консистентности #2"
  ],
  "branch_note": "куда свернула история после решения пользователя",
  "state_patch": {
    "tension": 0.55,
    "open_loops": ["строка"],
    "resolved_loops": ["строка"]
  },
  "suggested_directions": [
    "направление 1",
    "направление 2",
    "направление 3"
  ],
  "image_prompt_en": "cinematic illustration prompt in English (optional)"
}
```

Требования:
- `scene_text` всегда на русском.
- `tension` в диапазоне 0..1.
- `suggested_directions` — необязательное поле, но добавляй его, если пользовательский ввод расплывчатый или противоречивый.
- Не меняй базовые законы мира без явного пользовательского запроса.
