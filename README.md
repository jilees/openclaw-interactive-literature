# openclaw-interactive-literature

Отдельный репозиторий для интерактивной литературы в OpenClaw.

## Primary layout

- `AGENTS.md` — основной контракт/правила работы.
- `scripts/` — генерация текста и аудио.
- `references/` — контракт истории и пресеты.
- `memory/stories/` — память и прогресс историй.

## Compatibility note

Исторический путь `skills/interactive-literature/` сохранён как legacy mirror на период миграции.
